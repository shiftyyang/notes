### 2. 复习计划

#### JavaFamily：

* JAVA 基础核心串讲：操作系统、计算机网络、java 基础和新特性、设计模式
* JVM
* 并发和多线程
* 常用工具：JVM 排查工具 JMC、线上调试、git、linux 常用分析工具
* 数据结构算法
* 必会框架：spring、netty、RPC、ORM 框架 Mybatis

* 数据库：mysql 索引、锁，事务特性、隔离级别、sql 调优、分库分表
* 搜索引擎：es、lucene、canal、logstash
* 架构演进之路：one 到微服
* Docker 和 k8s 详解
* 开源项目
